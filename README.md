# 游戏规则

## 1. 棋盘结构：

- $27 \times 27$ 的大棋盘，分为 3 层九宫格嵌套
- **大层**：$3 \times 3$ 个大九宫格，每个 $9 \times 9$
- **中层**：每个大九宫格内 $3 \times 3$ 个中九宫格，每个 $3 \times 3$
- **小层**：每个中九宫格内 $3 \times 3$ 个小格子，每个 $1 \times 1$

## 2. 落子规则：

- 黑棋先行，第一步可在全盘任意位置落子
- 之后每一步必须落在上一步指定的小九宫格内
- **指定方法**：上一步坐标为 $(p,q)-(m,n)-(i,j)$  
  下一步必须落在 $(m,n)-(i,j)-(x,y)$ 的位置
- 如果指定的小九宫格已满，可在对应的中九宫格内任意落子
- 如果中九宫格也已满，可在全盘任意位置落子

## 3. 占领规则：

- 在 $3 \times 3$ 小九宫格内三连（横、竖、斜），则占领该小九宫格
- 占领的 $3 \times 3$ 小九宫格在中九宫格内三连，则占领该中九宫格
- 占领的中九宫格在大九宫格内三连，则获胜
- 同一区域可被双方同时占领

## 4. 胜利条件：

- 在 $3 \times 3$ 的大九宫格层面形成三连的一方获胜
